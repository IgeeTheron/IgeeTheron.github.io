Technology used to construct this CV

The web page is coded in HTML5, CSS3, and JavaScript. The Flexbox of CSS3 was my first option but after testing I found that the Grid of CSS3 fits better and is more flexible. For the background images, I wanted it to stand out less but still look good, so I used after effect code to add shadows to it.

The web page ‘Font Awesome’ I used to load in icons for my tabs, education, footer link and bullet points. After testing I found that inserting tabs is much more convenient than inserting more pages. I used knockout text and hard stop gradient to generate the animation on my email link. I also used a font grow and colour change with hover to give the GitHub link more personality when hovered upon. I made sure that the web page is responsive over all devices including portrait and landscape mode for phones and tablets.
