Technology used to construct this Cite

This website is written in HTML5, CSS3, and JavaScript. I used CSS3's Grid to create the tabs and layout. I used the CSS's Flexbox to structure the webpage into responsive columns. For the background images, I wanted it to stand out less but still look good, so I used after effect code to add shadows to it.

‘Font Awesome’ is used to load in icons. After testing I found that inserting tabs is much more convenient than loading new pages. I used knockout text and a hard stop gradient to generate the animation on the email link. I used a font grow technique and colour change with the hover selector to give the GitHub link more personality when hovered over. I used @keyframes to create a light loading animation. I also added an animation to fade away the loader once the page's content has been loaded. I used the Fetch API to get information on other websites to display on my website. I made sure that the website is responsive on all devices including portrait and landscape modes.
